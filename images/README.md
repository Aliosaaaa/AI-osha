# Istruzioni per i Loghi

## Come aggiungere i tuoi loghi al sito

### 1. Logo AI OSHA (Homepage)
- **Nome file:** `logo-aiosha.png`
- **Posizione:** Salva questo file in questa cartella `/images/`
- **Quale logo:** Il logo con scritta "AI OSHA" (secondo logo che hai caricato)
- **Dove apparirà:** Nella homepage, sopra il titolo principale

### 2. Logo Intelligenza Artificiale in Pratica (Pagina Corso)
- **Nome file:** `logo-corso.png`
- **Posizione:** Salva questo file in questa cartella `/images/`
- **Quale logo:** Il logo con scritta "INTELLIGENZA ARTIFICIALE IN PRATICA" (primo logo che hai caricato)
- **Dove apparirà:** Nella pagina corso, sopra il titolo principale

## Istruzioni Passo-Passo

1. Scarica le due immagini dal tuo computer o dalla chat dove le hai caricate
2. Rinomina il logo "AI OSHA" in: `logo-aiosha.png`
3. Rinomina il logo "Intelligenza Artificiale in Pratica" in: `logo-corso.png`
4. Copia entrambi i file in questa cartella: `/home/user/AI-osha/images/`
5. I loghi appariranno automaticamente nelle rispettive pagine!

## Dimensioni Consigliate

- Logo AI OSHA: max 500px di larghezza
- Logo Corso: max 600px di larghezza

(Le dimensioni si adatteranno automaticamente su dispositivi mobili)
